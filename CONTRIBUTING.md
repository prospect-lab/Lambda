# Contributing to Lambda
Lambda is an Open-Source project made for everyone, by everyone. It is the work of us humans and we would appreciate your help.

## Issues
If you are unsure whether you have found a bug, please consider asking in the [Discussions](https://github.com/prospect-lab/Lambda/discussions) section.
If the behavior you are seeing is confirmed as a bug or issue, it can easily be re-raised in the issues section.

## Filing issues
You need to gather some information before filing an issue. Those informations consist of:
1. What version of Lambda you've faced this issue in (ex. Version One or Commit ABCD1234)
2. Make sure that the issue doesn't exist in our Issues section or Discussions section.
3. What operating system and processor architecture are you using?
4. What did you do?
5. What did you expect to see?
6. What did you see instead?

For proposing changes, go to the Ideas channel in our Discussions section.

## Contributing code
Please make sure that you obey the rules in our [Code Of Conduct](./.github/CODE_OF_CONDUCT.md)
